新的规划

发表于 2016-05-18

The second try

既学斋之后，第二次起航，开启Startup之旅。这一次选择的方向更加的平易近人，从一个红海的市场进行切入，核心是Startup的服务对象是蓝海市场。

做了那么多工作，换了那么多公司，职涯起起伏伏屡见不鲜。同样对于品目繁多的创业项目而言，我觉得返璞归真，从最简单的、最能凝聚手中资源、最能有话语权的项目开始做最为舒服。

项目：4A信息服务公司

有意义的事情：
  1. 注册一家公司，为Startup正式扬帆起航
  2. 真正的销售处第一单，这个比什么都重要。对于之前的学斋网，很难销售的出去，也很难吸引投资者的注意。那么这次选择换一个思路，找一个成熟的市场，有大把的资源可以借鉴，通过打通渠道制造销售业绩，用实体推动项目的发展。
  3. 实现小团体的组建。之前的学斋，说白了就两个人摸黑前行，效率、思维、行动力都有很大问题，这次我想组建一个团队，主要是从大大结识的一帮伙伴，而这个项目都能让他们的特长有所发挥，虽然目前我还没有抛橄榄枝，但我隐约觉得我会有这个号召力。
  4. 实现自我价值的突破以及薪资上的突破。
需要坚持的事情：
  1. Worktile是我这次突发灵感想到的工作协调工具，我觉得坚持使用非常有必要，即使为了装逼。
  2. 既然这个项目为渠道而生，那么销售工作是一个非常艰辛的活，需要想尽办法的去发动资源去寻找。
  3. 坚持存活下去，发现蓝海市场，比如3D打印、VR视觉等一系列可以让我们在这个市场上活下去并获得盈利的切入口。
  4. Time & Hard Work。想要成功，勤奋的努力不可或缺，加油吧！
