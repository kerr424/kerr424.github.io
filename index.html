新的规划

发表于 2016-05-18

The second try

既学斋之后，第二次起航，开启Startup之旅。这一次选择的方向更加的平易近人，从一个红海的市场进行切入，核心是Startup的服务对象是蓝海市场。

做了那么多工作，换了那么多公司，职涯起起伏伏屡见不鲜。同样对于品目繁多的创业项目而言，我觉得返璞归真，从最简单的、最能凝聚手中资源、最能有话语权的项目开始做最为舒服。

项目：4A信息服务公司

有意义的事情：
  1. 注册一家公司，为Startup正式扬帆起航
  2. 真正的销售处第一单，这个比什么都重要。对于之前的学斋网，很难销售的出去，也很难吸引投资者的注意。那么这次选择换一个思路，找一个成熟的市场，有大把的资源可以借鉴，通过打通渠道制造销售业绩，用实体推动项目的发展。
  3. 实现小团体的组建。之前的学斋，说白了就两个人摸黑前行，效率、思维、行动力都有很大问题，这次我想组建一个团队，主要是从大大结识的一帮伙伴，而这个项目都能让他们的特长有所发挥，虽然目前我还没有抛橄榄枝，但我隐约觉得我会有这个号召力。
  4. 实现自我价值的突破以及薪资上的突破。
需要坚持的事情：
  1. Worktile是我这次突发灵感想到的工作协调工具，我觉得坚持使用非常有必要，即使为了装逼。
  2. 既然这个项目为渠道而生，那么销售工作是一个非常艰辛的活，需要想尽办法的去发动资源去寻找。
  3. 坚持存活下去，发现蓝海市场，比如3D打印、VR视觉等一系列可以让我们在这个市场上活下去并获得盈利的切入口。
  4. Time & Hard Work。想要成功，勤奋的努力不可或缺，加油吧！

金融公司的危机公关

发表于 2016-05-05

雪崩时，没有一片雪花觉得自己有责任！

出现公关危机

如今的金融行业，被爆已经成为日常。以致于原来对于此非常灵敏的媒体都对此丧失了兴趣。如今我目前的东家，正陷于此危机之中。

之前我已经经历过D公司的风波，对此也已经轻车熟路。今日X公司初起势头，先不论这波危机能否遏制，仅就控制手法和具体动作分析一下。

核心思想

  1. 透明度
  2. 集合力量
  3. 避免内讧
  4. 择机择时择地

具体分析

  • 危机不是一日造成的，出现延期兑付或是无法兑付的情况肯定是有所预知的。负责产品、运营、财务的同事肯定是最知道内部行情的，老板毕竟要担法律风险，也不会不重视。那么是要将这包火盖住，成立个紧急消防队还是将危难坦然处之，选择一条正确的解决方式告诉大家对此做好抵御准备？

  • 问题的关键点来了？究竟是否有解决这个危机的本质方法！（此点是核心，也是具体动作的核心假设。如果XXX公司完全玩的是非法集资、庞氏骗局，那么只要出现危机，没有任何的挽回余地；如果仅仅是资金出现缺口，需要时间来弥补，那么下面所描述讨论的方法或许还有点作用）

    • 出现危机后参与的人群可以简单分为两类：外部客户&内部理财经理

    • 对于外部客户：从门禁卡进行管控最为有效

    • 对于内部理财经理以及理财经理携客户的群体：以内部理财经理作为切入口，将后续行之有效的解决方法反复沟通；

    • 转移战场非常重要，尽量集中至统一会场，在危机前期公司一方还是拥有绝对的主动权，可以好好把握，争取扭转战局；

    • 切忌所有的沟通都是确切的、真实的、诚信的，否则会导致危机越滚越大；

  • 还有一点就是，对于没有经历过的人来说，出现危机，对于公司其他内部的工作人员影响极大，很多人一脸茫然，双眼无神。我觉得这就是内部透明度和提前预知的工作没有做到位。出现危机公关前，一定要让现有公司员工拧成一条绳，与公司共患难。否则，在危机的过程中，很多情况并不是客户把公司造成了多大的创伤，而是内部员工完成了对公司的致命一击。

最后，跟那些冲动的客户说一句：如果这个公司真的有起死回生的本事，留一条生路，说不定船到桥头自然直。如果死死相逼，上升到经侦、国家那个层面，你猜结果会怎样？

求之于势不责于人

发表于 2016-05-04

孙子兵法·兵势第五

故善战者，求之于势，不责于人，故能责人而任势。

关于鸡汤

最近看了两本书，一本是《Zero to one》，作者是创立Paypal，和钢铁侠Musk一起奋斗、公事过的男人，首位Facebook的外部投资人—Peter Thiel，一本是《创业唯艰》，作者也是网景公司的创始人之一，初期投资过Facebook、Twitter、Groupon的大牛。
两本书的主题只有一个Startup，最近整理了下思绪，有三个Level。

No. 1 高富帅

比如雷军，因为强大的背景，在投资人对小米手机完全不知情，甚至不知道是否要做手机的情况下，估值已达2.5亿，这是何等的霸气。

我去查阅了下雷总在金山以及在卓越的经历，才发现原来人家是中关村的元老级别的大人物。当然运气也是上乘，在金山起步，29岁就成为金山的北京总经理。相比我现在的状态，真心是痴人说梦话的节奏。

No.2 工人阶级

不管是Zero to one还是创业唯艰，成功创业学里的道路总是高不可攀，即使是工人阶级，从打工者出身的职业经理人。

  • 他们起始点均在大公司
  • 他们的创业思路久经市场的洗礼
  • 他们曾在高位并运用高位资源，结识或组件了Startup的合伙人/创业团队

其实，这条道路最适合我，也最适合大部分人吧。

No.3 屌丝

这一阶级主要来自知乎这种奇葩平台上，有些故事我真的很难相信是真的，但好像感觉真有那么一回事，因为这些奇葩经历或是成功故事都存在于三四线城市，环境不同这些招数或许真的管用。

但他们的草根案例也有两个特点：

  • 他们突出的更多的是勤奋、运气、果敢、执着
  • 他们的道路更难以复制或是令人不想复制

小结

思路慢慢清晰了，对自己的定位也越来越明确。

后续延伸两个话题这周继续理清：

孙子兵法（这也是我引用的那句话，用直白点的语言说就是：做风口里的猪）

自我销售 (我觉得这是我目前比较欠缺或是亟待加强的一点，尤其是我知道目前公司的CEO仅仅30岁时，更是激发了我对一点的反思)

四月结束

发表于 2016-04-29

不要用战术上的勤奋掩盖战略上的懒惰。

这句话曾在罗辑思维里面听到过（当时是在讲二战时期的日本），今天在看微信公众号时翻到一篇，重燃了很多念想。

关于四月

April是我的生日月，我对这个月份的感情很好，万物复苏，草绿萌萌。

虽然事务烦身，但仅仅是偶尔闲暇，快步走进静安寺公园，看着那壮硕繁茂的大树和平整新香的草地就会感觉到一种春天的气息，让人心旷神怡。

我不太喜欢风景的一草一木，也不想过多的描述具体的细节。用一句话来说明，四月的世界都是萌新的，是翻天覆地的，宛若有点踏入异乡的感觉，这点让我很是享受。

Hexo

今天的工作还算比较繁忙，也没什么太多的时间去学习和感受。

今天就谈谈一个老话题，一直想说但却没有太多切入口的主题：Hexo，就是建这个平台的终极武器。

学习写个人博客，这个大家都肯定熟悉，比如新浪博客、网易Lofter等等，但如今的《三千世界》，则是自己建站、自己管理的一个各人博客，感觉很装逼，很是喜欢。

先聊一聊建站的大致途径：

  • 基础：不管你是OS还是Win下，都得对终端有所了解（因为Bash、Github都是在这个代码狂框里运作的）
  • 工具：慢慢学习Markdown编辑语法，虽然说不难，但总归要写个两三篇文章才能顺手吧。而且有些较真的人选个Markdown的编辑器都要选半天（就是我…）
  • 调试：有了基础和工具，你还得会在终端Bash下进行调试。我在这个过程可是卡了三次，近一周多的时间，很是郁闷，谁让我没有那么多会编程又喜欢装逼的朋友呢
  • 坚持：博客最难的还是坚持，偶尔写两篇谁都可以做到，你说不是么？

startup

发表于 2016-04-28   |   分类于 随笔

浅薄的人才会相信运气和境遇……强者只相信因果。
——拉尔夫·艾默生

最近在看《Zero to one》这本书，感触颇多，加之工作上的一些事情对于Startup这件事又重燃心头。

关于工作

  • 在做学斋网创业的时候，明显的感觉到有份工资的可贵。尤其在看过那么多的资料，上海独特优厚的打工环境，让Startup的意愿一减再减。
  • 目前，整个金融的大坏境已经大不如前，明显感觉到泡沫即将爆破的痛楚。不想挣扎的企业选择跑路或缴械投降，还有力气再折腾的企业还梦想着转型或是被收购。但对于打工者而言，工作环境已经悄然发生着转变。尤其对于向往上升空间的人们来说，工资很可能是过眼云烟，公司说没就没，很有可能要白打两个月的工，甚至有可能触及法律风险。这笔买卖着实让我再度深深思考未来的走势方向。
  • 另外，这三年的时间基本的工作套路是跟老大。作为团队中的一份子去参与到工作之中，但对于这场大风浪即将来袭，每个人的地位、身份、所求全都不一致的话，很难同心协力。用最直白的话来说，有些人是风险厌恶，有些事风险喜好，还有些是风险文盲，你说这个团队在这大海啸即将到来的时候如何能做好工作？

关于Startup

从零到一的一个核心观点和思路就是，是从零到一而不是从一到N，这是有本质差别的。最直接的概括，就是指，你所要切入的市场是个完全竞争市场还是垄断市场，所收获的利益和回报是天差地别的。同时，如果仅仅是复制别人的成熟技术或是套路，本身就没有特殊的禀赋或资源去支撑或还原，更何谈创新和发展呢。

但有几个问题很有意思，我也在反思：

  • 不做永远是没有机会的；
  • 没有一个成功的企业发展的道路是相同的；
  • 我们除了模仿还能做什么；
  • 团队如何组建；
  • 细分市场如何切入；
  • 最小化可实施模型如何搭建；
  • 痛点如何寻找；
  • …………

这些问题都得慢慢思考咯！

关于起始

关于起始

这是最好的时代，也可能是最坏的时代。

The Aim of Half year 2016

转眼间就到了2016年的四月份，中间的工作状态起起伏伏，生活状态也伴随着怀孕的一切周边所牵扯动荡。

工作来讲

  1. 薪酬才是首要的，职业发展都要靠边站；
  2. 定位是模糊的，学习东西才是首要的；
  3. 未来的方向总体而言是要走杂+复合+国际+管理的路线，但具体的细节我也无法确定熟好熟劣，且走且看；
  4. 关于工作配合尽量保持不卑不亢的态度，不奢求认真做即可。当受委屈的时候，想想那句话，看看薪酬！；

生活来讲

  1. 首要的是把宝宝生下来，这件事可能是16年最最最重要的事情了，没有之一；
  2. 其次生下来还牵扯到一系列的事情，比如：去美国生产、谁来照顾、多和老婆说话、饮食、美国购物准备等等；
  3. 对自身而言，相对比较自律的我，对于健身希望在16年能够有着比较好的坚持，今年的目标就是在引体向上及双力臂上有所突破；
  4. 对于游戏而言，目前尽量控制，但很难压缩（我所希望的仅仅是2个手机App每天占用时间大概在20分钟左右，1个电脑游戏炉石每天占用时间大概20分钟）。我清楚有时希望仅仅是一种寄托，更多的还是回归现实吧；

朋友关系来讲

  1. 朋友之间需要时间来维系，总觉得自己身边的朋友太少，有种不知足的情节；
  2. 好好利用身边的资源，比如申哥办公室的茶我爱喝的很、玉兰香苑的羊肉串我也爱吃的很、杨浦五角场吃顿大的也未尝不可，那就当有这种想法的时候去找相应的群体呗；
  3. 工作的朋友关系想维系好难度很高，不用奢求；

学习来讲

  1. Python一定要在今年把它啃下来（本身想是在6月份用半年的时间把它搞定，现在看看是遥不可及了，当然这里也是我时间计算有误，实话说学Python的这段时间还是挺努力的）；
  2. 设计、编程、各种小软件、各种小技巧、知乎、站酷、喷嚏图卦多刷多刷；
  3. Kindle的读书时间尽可能保持，现在的难度就是我无法发现好书……；
  4. 英语必须要抓起来咯，毕竟娃子是美国人；

The End

杂乱的说了一大堆，总结几个关键字吧，工作-薪酬、生活-多万、朋友-时间、学习-Python。

还有，如果我能坚持写Markdown日记也是一个有趣的事！
